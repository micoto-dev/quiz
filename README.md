# まっぷるQUEST — 伊豆ダンジョン PoC

まっぷる記事を元にしたご当地クエストアプリのプロトタイプです。
「旅行体験そのものをゲーム化する」コンセプトを検証するためのPoC。

## コンセプト

- **5問の通常クイズ** — 伊豆の名所（修善寺・大室山・稲取・三島・堂ヶ島）の基礎知識
- **1問の現地ミッション** — 実際に現地の看板・石碑を読まないと解けない問題
- **ボス戦** — 天城のわさびに挑む最終問題
- **自治体PR枠** — 正解時に表示される、協賛スポンサー向けのPRカード

## 特徴

- 📱 スマホ最適化（縦長UI）
- 🗺️ 位置情報を使わない「自己申告型」現地ミッション設計
- 🎨 まっぷる風の和モダンデザイン（明朝体・金色のアクセント）
- 🎯 自治体協賛枠をUI内に組み込み済み

## 技術スタック

- React 18（CDN読み込み）
- Babel Standalone（JSXのブラウザ実行）
- Single HTML file（ビルド不要）

## 想定利用シーン

- 5/9 伊豆町長面談用デモ
- 自治体への協賛提案素材
- 地域観光協会向け企画提案

## ステータス

Proof of Concept（PoC）段階。本番導入時はビルド版への移行を予定。
